Getting Started

Welcome

Thanks for purchasing our template :)

It is our great pleasure that you purchased our template. And we want to tell you that you don't need to write any codes for this template. Just follow some instructions for customizing the template.

Requirements

  • Make sure to install nodejs to run this template. You can download them by the following link below https://nodejs.org

Installation

Unzip the archive and find out the folder "buyer-file"

You must have to install packages. You can do this by running 'npm install' from the "buyer-file" folder of your project to install all the necessary dependencies.

  1. npm install
  2. npm start
  3. Navigate to http://localhost:3000/
