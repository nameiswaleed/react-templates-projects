They are exclusively designed to help out online product listing websites with professionally designed templates. It offers a list of category-specific pages and multiple variations pages for all types of learning management platforms.

1. 2. Installation Guide

React.js

  • Make sure to have React CLI installed globally & running on your computer. If you already have installed React CLI on your computer, you can skip this step.

npm install -g create-react-app

Nodejs

  • Make sure to have Node.js installed & running on your computer. If you already have installed Node on your computer, you can skip this step if your existing node version is 18.13.0.
  • After you finished with the above steps, you can run the following commands into the terminal/command prompt from the root directory (listee/reactjs/ ) of the project to run the project locally or build for production use:
  • This would install all the required dependencies in the node_modules folder.

npm i --legacy-peer-deps

  • Runs the project locally, starts the development server and watches for any changes in your code, including your HTML, Javascript, css, etc.

npm start

  • Now you can run this Template in any web browser from your PC or Laptop.

http://localhost:3002

  • Generates a /dist directory with all the production files.

npm run build

  • reactjs/ is a folder name, you can change as per your wish how you used the folder
